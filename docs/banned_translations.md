# Traducciones prohibidas / a evitar

Lista negra obligatoria. Ningún agente debe producir esto.

## 1. Caracteres prohibidos (fuente no los soporta)
- `á é í ó ú` → usar vocal sin acento.
- `ñ` → `n` (ej. `nino`, `senor`, `ano`).
- `¿` `¡` → NO usar. Pregunta/exclamación solo con `?` / `!` al final.
- `"` `«` `»` `…` `—` (raya larga) `ü` → prohibidos. Usar `-`, `.`, `'`.

## 2. Españolismos (variante España) — PROHIBIDOS
| Prohibido (España) | Usar (latino) |
|---|---|
| vosotros / os / vuestro | ustedes / les / su |
| tío / tía (coloquial) | compa / men / nene |
| flipar, molar, currar | alucinar, gustar, trabajar |
| vale (de acuerdo) | esta bien / okay |
| gilipollas | tarado / idiota |
| coche | auto / carro |
| ordenador | computadora |
| chaval | chico / muchacho |

## 3. No traducir (mantener en idioma original)
- Nombres propios de personajes (Mark, Yukino, Nanjo, Kandori...).
- Nombres de Personas y demonios.
- Nombres de hechizos japoneses (Agi, Bufu, Zio, Garu, Dia, Mudo, Hama...).
- Onomatopeyas (Gyahahaha, Heeheehee, yip, ring, crackle, splat...).
- "Persona", "SEBEC", "Arcana".

## 4. Calcos / errores comunes a evitar
- No traducir literal "spell card" como "tarjeta de hechizo" → es `carta de hechizo` / `carta`.
- No traducir "human" despectivo como "ser humano" formal en boca de demonio → `humano`.
- No usar "negociación" con acento → `negociacion`.
- No inflar la longitud: evitar perífrasis ("hacer una pregunta" → "preguntar").
- No cambiar el registro: un demonio grosero no habla formal y viceversa.

## 5. Tokens
- PROHIBIDO traducir, mover, duplicar o borrar cualquier `<....>`.
- PROHIBIDO traducir la palabra dentro de `<0056>palabra<0056>` si es onomatopeya.
